# Daily Problems for 2026.06.15

1. Zip two arrays.
2. Zip two strings.
3. Zip N arrays.
4. Zip N strings.

Above problems have common property where the resultant array/string is as long as the shortest array or string.

For problems below, you will pass a default null-value to the function which will be used when the shorter array/string value is not available. The resultant array/string is as long as the longest one.

5. Zip two arrays with null-value.
6. Zip two strings with null-value.
7. Zip N arrays with null-value.
8. Zip N strings with null-value.

And the problems become tougher:

9. Zip two streams, until one of them signals end.
10. Zip two streams, until all of them end. (use null-value).
11. Zip N streams, till one ends.
12. Zip N streams, till all end. (null-value)

And this is just weird:

13. Zip two columns in a SQL table. Use null-value.
